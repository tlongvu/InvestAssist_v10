# Testing Patterns

**Analysis Date:** 2026-03-03

## Test Framework

**Runner:**
- PHPUnit 10+
- Config: `phpunit.xml`

**Assertion Library:**
- Standard PHPUnit assertions

## Test File Organization

**Location:**
- `tests/` directory (separate from source)

**Naming:**
- `*Test.php`

## Test Types

**Feature/Integration Tests:**
- Used to test full application flows (Auth, Dashboard loading)

**Unit Tests:**
- Minimal presence, focus is on integration.
